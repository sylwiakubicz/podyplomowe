## Opis projektu
Program służy do tworzenia obiektów o klasie Cat. 
Obiekty te potrafią mówić "meow", jeść i możemy przedstawić naszego zwierzaka w oparciu o następujące atrybuty:
* name
* color
* age
* hungry
